### Dockerの基本操作

```
基本的なDockerコマンドの説明、ハンズオン(Nginxあたりを使用？)
・HTTPサーバを起動(run –d)
・起動状態の確認(ps)
・ログの確認(logs -f でログを見ながらWebページ表示)
・コンテナに入って新規Webページを作成、アクセス(exec)
・コンテナ外で新規Webページを作成、ドキュメントルートに配置してアクセス(cp)
・HTTPサーバを停止(stop → ps -a)
・HTTPサーバ削除(rm → ps -a)
```

### イメージ・composeの作成

```
・イメージ作成
事前にWebページを作成
DockerfileでNginxのドキュメントルートにWebページをコピーしたイメージを作成
起動してアクセス確認

・compose作成
Nginx,httpdをUP、Nginxをproxy、httpdをWebサーバとして、
Nginx→httpdと連動するdocker-compose.ymlを作成
```
